## ขั้นตอนการติดตั้ง 
1. โหลด project ลงเครื่อง
``` bash
git clone https://github.com/Witchaz/mini_project_ML.git
```
2. สร้าง virtual environment
``` bash
python -m venv venv
```
3. ติดตั้ง package  ที่ใช้ 
``` bash
pip install -r requirements.txt
```
